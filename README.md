# Hyperliquid 跟单系统

> **只读监控面板 + 本地跟单机器人**  
> 私钥永远只在本地出现，绝不上传任何服务器

## 架构

```
┌─────────────────────────────────────┐
│  Vercel 部署 (只读)                  │
│  https://frontend-chi-three-72.vercel.app  │
│  ├── 前端面板 (React)               │
│  ├── API (查看参考账户数据)           │
│  └── 无私钥、无交易功能               │
└─────────────────────────────────────┘
         ↑ 查看参考账户持仓
         ↓ Web / API
┌─────────────────────────────────────┐
│  本地运行 (跟单机器人)                │
│  ├── node src/index.js              │
│  ├── 私钥仅在本机内存解密             │
│  ├── 自动跟单引擎                    │
│  └── 手动平仓                        │
└─────────────────────────────────────┘
```

## 功能

- **参考账户扫描** — 实时扫描指定账户的持仓、挂单、交易记录
- **跟单机器人** (本地) — 按比例自动同步持仓
- **手动平仓** (本地) — 单币种 / 全部平仓
- **私钥安全** — AES-256 加密，仅内存解密签名

## 💻 本地运行（跟单机器人）

### 1. 安装依赖
```bash
cd backend && npm install
```

### 2. 配置私钥
编辑 `backend/.env`:
```
YOUR_PRIVATE_KEY=0x你的私钥
```
> 私钥**绝不在代码中硬编码**，`.env` 已被 `.gitignore` 排除

### 3. 启动跟单机器人
```bash
cd backend && npm run dev
```

### 4. 打开本地面板
```bash
cd frontend && npm run dev
```
访问 http://localhost:5173

## 🔐 私钥安全说明

| 项目 | 说明 |
|------|------|
| 存储 | AES-256 加密，密码解锁 |
| 网络 | 永不离开本机 |
| 签名 | 仅在内存中解密，签名后立即清除 |
| Vercel | 完全不涉及私钥 |
| 建议 | 使用独立跟单钱包，小额资金测试 |

## 技术栈

- **后端**: Node.js + Express + WebSocket + Ethers.js
- **前端**: React + Vite + TailwindCSS
- **链**: Hyperliquid (Arbitrum)
- **部署**: Vercel (只读部分)
